Tips/How to use

  • Drag'n'drop an image you want to process
  • Wait several seconds to process the image
  • Move two sliders to correct the image for your taste
  • Save the image in a suitable format
  • Use the mouse wheel or buttons to zoom in/out
  • Use the double left click or button to reset the view
  • Hold the left click and move to drag a selected image region

Version 1.0.1
This is a free open-source software written in C++ and released under the GNU GPLv3. The most recent version of the application is available on this website. The source code of the application and the library is available on GitHub.

Description
A human eye differs from a digital camera a lot. There are some biological algorithms our eyes implement to receive a better and clearer view of what we see. Digital cameras may not simulate such algorithms out-of-the-box (they use some basic digital algorithms, however). This software is aimed to simulate some of human eye algorithms such as chromatic adaptation and local lightness adaptation to receive a "more natural" view of images we get from digital cameras. This project is based on, but not equal to implementation of iCAM06 and CAM16 color appearence models.
