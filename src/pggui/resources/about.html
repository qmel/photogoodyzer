Tips/How to use

  • Drag'n'Drop an Image you want to process
  • Wait several seconds to process the Image
  • Move two sliders to correct the Image for your taste
  • Save the Image in suitable format
  • Use Mouse Wheel to zoom in/out
  • Double Left Click to reset the view
  • Hold Left Click and move to drag selected Image region

Version 1.0.0
This is a free opensource software written in C++ and released under GNU General Public License. Most recent version of the application is available on this website. Source code of the application and the library is available on GitHub.

Description
A human eye differs from a digital camera a lot. There are some biological algorithms our eyes implement to receive a better and clearer view of what we see. Digital cameras may not simulate such algorithms out-of-the-box (they use some basic digital algorithms too, however). This software is aimed to simulate some of human eye algorithms such as chromatic adaptation and local lightness adaptation to receive a "more natural" view of images we get from digital cameras. This project is based on, but not equal to implementation of iCAM06 and CAM16 Color Appearence models.
